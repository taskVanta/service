Task Vantra API Documentation

Endpoints

Base URL

https://service-q9nm.onrender.com/api

POST /api/users/signin

User login using curl

curl -X POST https://service-q9nm.onrender.com/api/users/signin \
-H "Content-Type: application/json" \
-d '{"email":"user@example.com","password":"yourpassword"}'

User login

https://service-q9nm.onrender.com/api/users/signin

Request Body

   
{
    "email":"user@example.com",
    "password":"yourpassword"
}

POST /api/users/signup

Create a new user account using curl

curl -X POST https://service-q9nm.onrender.com/api/users/signup \
-H "Content-Type: application/json" \
-d '{"name":"John","email":"john@example.com", "password":"yourpassword"}'

User login

https://service-q9nm.onrender.com/api/users/signup

Request Body

   
{
  "name": "John",
  "email": "john.doe@example.com",
  "password":"hello@123"
}

PUT /users/{id}

Update an existing user by ID.

https://service-q9nm.onrender.com/api/users/TV0001

Request body:

{
  "name": "Updated Name",
  "email": "updated@example.com"
}

Response:

{
  "id": "TV0001",
  "name": "Updated Name",
  "email": "updated@example.com",
  "role": "user",
  "created_at": 1760618536804,
  "updated_at": 1760619536804
}
